## 框架

本網頁使用 Vue.js 框架構建。

## 開始使用

若要在本機上運行此網頁，請按照以下步驟進行：

1. 下載此存儲庫：
   `git clone https://github.com/ckmsc39th/ckmsc39th.github.io.git`
2. 安裝依賴軟體：
   `yarn`
3. 運行伺服器：
   `yarn dev`
4. 在瀏覽器中打開 `http://localhost:8080/`以查看網頁。
